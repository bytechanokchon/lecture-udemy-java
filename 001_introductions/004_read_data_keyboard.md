# Read data from Keyboard
วิธีการอ่านข้อมูลจากคีย์บอร์ดในภาษา Java

ใน Java เราจะใช้คลาสที่ชื่อว่า Scanner สำหรับการอ่านข้อมูลจากแหล่งที่มาต่าง ๆ
โดยคลาส Scanner มีแหล่งที่มาจาก utils package 
โดยวิธีการใช้งาน Scanner มีดังนี้

    Scanner sc = new Scanner(System.in);

System.in ใช้สำหรับรับ input เข้ามาจาก keyboard

โดยเราสามารถอ่านข้อมูลจากคีย์บอร์ดด้วยชนิดข้อมูลต่าง ๆ ได้ดังนี้
1. nextInt(); อ่านข้อมูลในรูปแบบเลขจำนวนเต็ม
2. nextFloat(); อ่านข้อมูลในรูปแบบเลขจำนวนเต็มพร้อมทศนิยม
3. nextDouble(); อ่านข้อมูลในรูปแบบเลขจำนวนเต็มพร้อมทศนิยม
4. next(); อ่านข้อมูลในรูปแบบของข้อความ
5. nextLine(); อ่านข้อมูลในรูปแบบของข้อความ โดยอ่านทั้งบรรทัด (รวมถึงเว้นวรรคด้วย)
6. nextByte();
7. nextShort();
8. nextLong();
9. nextBoolean();
10. hasNextInt(); ใช้ตรวจสอบว่าข้อมูลที่กรอกเข้ามาเป็นเลขจำนวนเต็มหรือไม่
11. hasNextFloat(); ใช้ตรวจสอบว่าข้อมูลที่กรอกเข้ามาเป็นเลขจำนวนเต็มพร้อมทศนิยมหรือไม่
