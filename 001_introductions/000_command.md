# Command

### javac -encoding UTF-8 <file_name.java>
ใช้สำหรับแปรงชุดคำสั่ง java (.java) ไปเป้น byte code (.class)

    javac -encoding UTF-8 Program.java

### java <file_name>
ใช้สำหรับรันชุดคำสั่ง byte code

    java Program

### javap <package_name>
ใช้สำหรับดูว่า package ที่ระบุ มี method อะไรให้ใช้บ้าง

    javap java.util.Scanner
