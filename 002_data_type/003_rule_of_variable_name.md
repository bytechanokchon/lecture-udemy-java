# Rule of variable name
1. ชื่อตัวแปรเหมือนกัน แต่ตัวเล็กตัวใหญ่ต่างกัน ถือว่าเป็นคนละตัวแปร
2. ชื่อตัวแปรสามารถมี ตัวอักษร ตัวเลข แต่ไม่มีตัวอักษรพิเศษยกเว้น _ และ $
3. ชื่อตัวแปรต้องขึ้นต้นด้วย ตัวอักษร หรือ _ หรือ $ เสมอ
4. ชื่อตัวแปรจะต้องไม่ใช่คำต้องห้ามหรือคำสงวน (ชื่อที่่ใช้อยู่แล้วในคอมไพเลอร์ เช่น int)
5. ชื่อตัวแปร สามารถตั้งชื่อที่ความยาวได้ไม่จำกัด
6. ใช้ camel case
