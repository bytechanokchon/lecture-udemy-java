# Data type in Java
โปรแกรมมักจะจัดเก็บข้อมูลไว้ในหน่วยความจำชั่วคราว (ram) ในระหว่างการทำงาน โดยเราจะเก็บข้อมูลไว้ในตัวแปร (variable) เพื่อที่เราจะรู้ได้ว่าตัวแปรนั้นจัดเก็บข้อมูลชนิดอะไรอยู่ เราจึงต้องระบุชนิดของข้อมูลให้กับตัวแปรด้วย 

ชนิดข้อมูลถูกแบ่งออกเป็น 2 กลุ่มหลัก ๆ คือ

1. Primitive Data Type

## Primitive Data Type
เป็นชนิดข้อมูลพื้นฐานใน Java ซึ่งกำหนดไว้ในคอมไพเลอร์ โดยแบ่งชนิดออกเป็นดังนี้

- **Interger** จัดเก็บเลขจำนวนเต็ม
    - byte จัดเก็บข้อมูลได้ 1 byte
    - short จัดเก็บข้อมูลได้ 2 byte
    - int จัดเก็บข้อมูลได้ 4 byte
    - long จัดเก็บข้อมูลได้ 8 byte
- **Floating Point** จัดเก็บเลขจำนวนเต็มพร้อมทศนิยม
    - float จัดเก็บข้อมูลได้ 4 byte
    - double จัดเก็บข้อมูลได้ 8 byte
- **Char** จัดเก็บตัวอักษรตัวเดียว
    - char จัดเก็บข้อมูลได้ 2 byte
- **Boolean** จัดเก็บค่าจริงหรือเท็จ
    - boolean ขึ้นอยู่กับระบบปฏิบัติการ

*ยิ่งใช้จำนวน byte เยอะ ก็ยิ่งใช้แรมและซีพียูเยอะในการเก็บและประมวลผลข้อมูล*
*1 byte = 8 bit*
