# What are Variable ?

ตัวแปร คือ ชื่อที่เรากำหนดให้กับข้อมูลที่เราจัดเก็บไว้บนหน่วยความจำ เช่น

    byte age = 40;

จากข้อมูลดังกล่าง
- b คือ ชื่อตัวแปร
- byte คือ ชนิดข้อมูลของตัวแปร
- 40 คือ ข้อมูลที่เราจัดเก็บไว้ในหน่วยความจำ

โดยตัวแปรจะถูกแบ่งเป็น 2 ส่วน
- byte b จะเรียกว่า declaration
- = 5; จะเรียกว่า initialization

ข้อมูลของแต่ละตัวแปรจะใช้หน่วยความจำเท่าไหร่ในการเก็บ ขึ้นอยู่กับชนิดตัวแปรที่เราระบุให้
